# NextJS + NextAuth.js + Prisma + GraphQL

# Install Packages

```bash
npm install
```

# Set up environment variable

```bash
cp .env.example .env

# Then edit .env
```

# Push schema to database

```bash
npx prisma db push
```

# Generate TypeScript Types

```bash
npm run codegen
```

# Start the app

```bash
npm run dev
```
